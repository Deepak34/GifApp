## Features

View trending gifs. Search by name. Favourite specific gifs. 

Includes pagination for efficient loading. 

## Architecture

Uses MVVM design pattern and RXSwift

## API Reference

GiphySwift, FLAnimatedImage, RxSwift, RxCocoa
